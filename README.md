# portal-cctv-journal

本项目用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖后端服务，可直接通过浏览器打开或托管于任意静态文件服务器。

## 项目简介

portal-cctv-journal 是一个静态页面归档仓库，主要存放各类独立编写的 HTML 文件。每个页面均为自包含的文档，不关联任何特定域名或网站。本仓库旨在提供一个整洁、可浏览的页面集合，方便查阅与引用。

## 目录说明

```
portal-cctv-journal/
├── index.html          # 项目入口页面（可选）
├── pages/              # 存放所有归档的 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有独立 HTML 页面，文件名应具有辨识度。
- `index.html` 可作为导航页，列出所有页面链接（非必需）。

## 页面归档说明

- 每个 HTML 页面均为独立文档，包含完整的样式与脚本。
- 页面间无相互依赖，可单独打开。
- 归档内容可能包括说明文档、工具页面、可视化展示等。
- 页面标题、描述及内容以实际文件为准。

## 维护说明

- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并保持命名规范。
- 不建议在页面中引用外部资源（如图片、脚本）以免失效；如有需要，建议使用相对路径或内联。
- 如需更新页面内容，直接替换对应文件即可。
- 本仓库不跟踪具体来源或用途，仅作为静态页面存档。
